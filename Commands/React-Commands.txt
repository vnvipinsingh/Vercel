Steps to create react app
1. Create react app
    npx create-react-app star-war-app
2. Change to the directory star-war-app
    cd star-war-app
3. Start the react app
    npm start

Steps to install bootstrap
1. Add bootstrap
    npm install bootstrap
2. Add css to src/index.js
    import 'bootstrap/dist/css/bootstrap.min.css';
